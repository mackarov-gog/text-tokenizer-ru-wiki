#### Коэффициенты для разных стилей текста

- **Художественный**:  
  - Коэффициент: $4{,}00$  
  - Символов: 116  
  - Токенов: 29  

- **Научный**:  
  - Коэффициент: $4{,}48$  
  - Символов: 130  
  - Токенов: 29  

- **Разговорный**:  
  - Коэффициент: $4{,}00$  
  - Символов: 48  
  - Токенов: 12  

- **Технический**:  
  - Коэффициент: $2{,}00$  
  - Символов: 92  
  - Токенов: 46
